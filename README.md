# WorkAdaptive Content

Jekyll-powered website for WorkAdaptive - AI + IDE training for data professionals.

## Site Structure

- **Homepage**: Value proposition, offerings, recent articles
- **Articles**: Blog posts and thought leadership
- **Services**: BI to AI Cohort, 1-on-1 implementation

## Local Development

1. Install Ruby and Jekyll
2. Run `bundle install`
3. Run `bundle exec jekyll serve`
4. Visit `http://localhost:4000/content/`

Live site: https://workadaptive.github.io/content/
